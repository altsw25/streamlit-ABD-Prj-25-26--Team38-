**Краткое описание:** Диабет — одно из наиболее распространенных хронических заболеваний, и своевременное выявление риска его развития имеет огромное значение для профилактики и лечения. В этом проекте будет проведено исследование медицинских данных женщин народности Пима, чтобы определить, какие факторы сильнее всего связаны с наличием диабета. Для выволнения данного проекта проводится анализ данных, статистические тесты и строится модель машинного обучения, способная прогнозировать вероятность заболевания.
